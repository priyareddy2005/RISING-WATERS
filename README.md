# 🌊 Rising Waters – Flood Prediction System

## 📌 Project Overview

Rising Waters is a machine learning-based flood prediction system developed to predict the possibility of flooding using environmental and weather-related parameters. The project aims to provide an early warning mechanism that can help reduce flood-related risks and support better disaster management.

---

## 🎯 Objectives

- Predict the likelihood of floods using machine learning.
- Analyze weather and environmental data.
- Provide an easy-to-use web interface for users.
- Assist in early flood preparedness and decision-making.

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/Rising-Waters.git
```

### Navigate to the project folder

```bash
cd Rising-Waters
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
